## 목적 추출 데이터
- "이름"의 "해당 날짜"의 "출결 상태"(셀 내용)

## 시트 선정 규칙
1. `site.md`의 추출 포맷에서 날짜 추출
2. 해당 날짜와 동일한 날짜(년, 월)를 가진 시트 선정
- 시트 이름 규칙 : `YY년 M월`

## 추출 컬럼 규칙
- 선정 시트 내 동일 날짜 열 찾아서 선정
    - 날짜 형식 : `YYYY. M. D`

## 추출 범위 규칙
1. 추출 컬럼 내 내용이 있는 행을 추출
2. 같은 행 A열 데이터와 페어로 반환
- A열 데이터 형식 : `이름`
- 추출 컬럼 데이터 형식 : `"텍스트"`
- 반환 데이터 : `[{"이름": "텍스트", "출결 상태": "텍스트"}, ...]`
    - 예시 : `[{"이름": "홍길동", "출결 상태": "지각"}, ...]`
