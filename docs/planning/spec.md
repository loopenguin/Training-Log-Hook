## 7. 핵심 기능 (웹 앱 대신 자동화 스크립트)

| # | 기능명 | 설명 | 입력 | 출력 | 우선순위 |
|---|--------|------|------|------|---------|
| F1 | 사이트 크롤링 | 특정 사이트 자동 로그인 후 갱신 버튼 클릭 및 페이지 데이터 추출 | - | 추출된 원시 데이터 | P0 (필수) |
| F2 | 구글 시트 연동 | 타겟 Google Spreadsheet의 문서 내용 읽어오기 | 구글 API 인증 키 | 시트 파싱 데이터 | P0 (필수) |
| F3 | 데이터 가공 로직 | 수집된 양측(사이트+시트)의 데이터를 요구 수식/형식에 맞춰 변환 | F1, F2 데이터 | 전송용 포맷 문자열 | P0 (필수) |
| F4 | 결과 전송 알림 | MVP 목적: 가공 완료된 최종 데이터를 디스코드 채널로 발송 | 디스코드 웹훅 주소 | 메신저 메시지 발송 | P0 (필수) |
| F5 | 상세 오류 추적 및 보고 | "페이지 크롤링 / 구글 시트 연동 / 데이터 가공" 중 어디서 에러가 났는지 구체적 지점을 검출하여 보고 | 구간별 예외 상황 (try-catch) | 발생 위치(구간)가 포함된 상세 에러 알림 | P1 (중요) |

## 8. 데이터 흐름도

```
[GitHub Actions 크론 트리거]
         ↓
[Playwright 특정사이트 크롤링] & [Google Sheets API 데이터 로드]
         ↓
    [데이터 병합 및 가공]
         ↓
  [Discord 웹훅 발송 (MVP)]  → (향후: 필요 사이트 시스템으로 직접 POST/입력 전송)
         ↘ (중간 실패 시)
    [Discord 통신 장애/에러 알림 발송]
```

## 9. 외부 API 의존성

*이 프로젝트는 자체 API 엔드포인트를 열지 않으며 외부 서비스 API만 활용합니다.*

| API / 라이브러리 | 용도 | 권한 방식 |
|----------------|------|----------|
| Playwright | 사이트 화면 제어 및 DOM 데이터 추출 | 계정 ID/PW 환경변수 |
| Google Sheets V4 | 스프레드시트 데이터 연동 | Google Service Account Key JSON |
| Discord Webhook | 결과 전송 및 상태 모니터링 | 단일 Webhook URL 환경변수 |

## 10. 빌드/배포 환경

| 항목 | 정의 |
|------|------|
| **실행 주체** | GitHub Actions (`ubuntu-latest` 워크플로우) |
| **트리거 조건** | `on.schedule.cron` (매일 한국시간 17시 30분) 및 `on.workflow_dispatch` (수동 On/Off 및 워크플로 즉시 실행) |
| **환경 변수 관리** | GitHub Secrets 저장소에 암호화 보관 (크롤링 대상 접속 정보, 구글 API 키, 디스코드 웹훅 URL) |

## 11. 환경 변수 및 사용자 설정 (Environment Variables)

이 자동화 파이프라인이 정상 동작하기 위해 사용자가 반드시 제공해야 하는 변수 목록입니다.
구조에 대한 이해와 발급 방법, 깃허브 시크릿 등록 방법 등 구체적인 사용법은 `docs/ENV_SETUP.md` 문서를 참조하세요.

| 변수명 (Key) | 설명 | 입력 예시 / 획득처 |
|-------------|------|-------------------|
| `TARGET_SITE_URL` | 접속할 목표 사이트의 로그인 페이지 또는 데이터 페이지 URL | `https://example.com/login` |
| `TARGET_SITE_ID` | 목표 사이트 로그인을 위한 아이디 | `admin_user` |
| `TARGET_SITE_PW` | 목표 사이트 로그인을 위한 비밀번호 | `password123!` |
| `GOOGLE_SHEET_ID` | 읽어올 구글 스프레드시트의 주소(ID) | `1BxiMVs0XRYFgwnTE...` |
| `GOOGLE_SERVICE_ACCOUNT` | 구글 API를 사용하기 위한 서비스 계정 JSON (Base64 인코딩) | GCP(Google Cloud Platform) 서비스 계정 메뉴 |
| `DISCORD_WEBHOOK_URL` | 작업 결과를 전송받을 디스코드 채널의 웹훅 주소 | `https://discord.com/api/webhooks/...` |
